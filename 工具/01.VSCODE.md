# Visual Studio Code

## 下載與安裝

1. Windows 上安裝 VS Code
    - Google `VS Code` 下載
    - 執行安裝程式：
        1. 勾選 `加到 PATH （需要重新啟動後生效）
        2. 勾選 `將 'Open with Code' 操作加到 Windows 檔案總管*檔案*操作功能表`
        3. 勾選 `將 'Open with Code' 操作加到 Windows 檔案總管*目錄*操作功能表`
        4. 完成安裝
    - 啟動
2. 在 macOS 上安裝 VS Code
    - Google `VS Code` 下載 (選擇 Intel 或 Apple Silicon)
    - 執行安裝程式：
        1. 拖曳到你的「應用程式」(Applications) 資料夾中
        2. 啟動
        3. 按下 `Cmd` + `Shift` + `P` (或 `F1`) 開啟 `命令選擇區`
        4. 輸入 `shell command` 選擇 `Shell Command: Install 'code' command in PATH`
3. 在 Ubuntu 上安裝 VS Code
    - Google `VS Code` 下載，選擇 「.deb (64 bit)」(適用於 Debian/Ubuntu)
    - 執行安裝程式：
        1. `sudo dpkg -i <你下載的檔案名稱.deb>`
        2. 相依性問題，接著執行 `sudo apt-get install -f`
    - 啟動
  
## 延伸套件

1. 基礎環境類 (讓 VS Code 更好用)
    - Chinese (Traditional) Language Pack for Visual Studio Code
      - 功能： 這大概是你的首要之選！它會把 VS Code 的介面 (選單、按鈕等) 全部變成你我熟悉的繁體中文，上手更容易。
    - Material Icon Theme
      - 功能： VS Code 預設的檔案總管中，所有資料夾和檔案圖示都長得差不多。安裝這個套件後，它會根據檔案類型 (例如 .js, .css, .py, .md) 顯示不同、美觀的圖示。
      - 為什麼好用： 你能一眼就分辨出檔案類型，找檔案快非常多！
    - Prettier - Code formatter
      - 功能： 這是目前最受歡迎的「程式碼格式化工具」。它不管你程式碼寫得多亂 (例如縮排亂七八糟、分號亂加)，只要你一存檔，它「唰」的一聲，自動幫你排版得整整齊齊。
      - 為什麼好用： 維持程式碼風格一致性是專業開發者的基本功，它能幫你省下大量手動排版的時間。
  
2. 網頁開發類
    - Live Server
      - 功能： 如果你是要寫網頁 (HTML/CSS/JavaScript)，這是必裝！它會在你的電腦上啟動一個本地伺服器，並在瀏覽器中打開你的網頁。
      - 為什麼好用： 最棒的是，當你修改 HTML 或 CSS 並存檔時，它會自動重新整理瀏覽器！你不用再手動按 F5 了，修改立刻看得見。
    - Live Share
      - 功能： 這是微軟官方推出的「即時協作」工具。你可以把你的 VS Code 編輯器「分享」給別人，邀請他們進來同時編輯同一份程式碼，就像「程式碼版的 Google Docs」。
      - 為什麼好用： 非常適合找同學、朋友一起寫作業，或請別人遠端幫你看 code 解 bug。
    - ESLint
    - Auto Rename Tag
    - IntelliSense for CSS class names in HTML
    - Git History

3. AI 類
    - Markdown All in One
    - Github Copilot、GEMINI、CHATGPT
