# Gemini 安裝

## 安裝 `NodeJs`

- Windows： 下載安裝檔並執行 https://nodejs.org/en/download
- MacOS： `brew install node`
- Linux： `sudo apt install nodejs`
- `node -v`
- `npm -v`

## 安裝 `Gemini Cli`

``` bash
npm install -g @google/gemini-cli
gemini
```

## 更新 `Gemini Cli`

``` bash
npm update -g @google/gemini-cli
```
