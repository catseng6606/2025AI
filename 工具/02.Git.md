# Git

## 下載與安裝

1. Windows
    - [Git-Scm](https://git-scm.com/) Download for Windows
    - `git --version`
2. MacOs
    - `Xcode Command Line Tools`
    - `brew install git` 升級 `brew upgrade git`
    - `git --version`
3. Ubuntu
    - `sudo apt update`
    - `sudo apt install git`
    - `git --version`

## 複製專案 (Clone)

- `git clone [URL]` 複製遠端儲存庫

## 設定個人資訊

- `--global` 全域，沒有此參數為非全域
- `user.name` 使用者
- `user.email` 使用者 email

1. `git config --global user.name "UserName"`
2. `git config --global user.email "Email@example.com"`

## 例外 `.gitignore`

``` .gitignore
# 忽略所有 .log 檔案
*.log

# 忽略 node_modules 資料夾 (非常重要！)
node_modules/

# 忽略 Mac 系統檔
.DS_Store

# 忽略機密環境檔
.env
```

## 初始化

- `git init`
- 產生 `.git` 資料夾

## 存檔

- `git add .`
- `git add {filename}`
- `git commit -m "這裡寫你的提交訊息"`

## 雲端同步

- 設定遠端 `git remote add origin [URL]`
- 上傳 `git push`
- 下載 `git pull` 與 `git fetch`
- `git pull` = `git fetch` + `git merge`
