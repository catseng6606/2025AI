# 使用 Github Pages

## Github

1. 登入
2. 個人首頁
3. 進入 `Repositories` (左上第二個)
4. 新增 `Repository`
   - 輸入 `Repository name`，例如 `hello` 
   - 確認 `Public`
   - `Create Repository`
5. 進入 `hello` Repository 選擇 `Settings`
6. 點選 `upload an existing file`
7. 上傳檔案 `index.html` 後點選 `Commit changes`
8. 後續可點選 `add file` 上傳
9. 網站連結：https://{user}.github.io/{hello}/index.html
